# Building Energy Simulation & AI Optimization

Energy simulation and AI-based optimization project developed for the **ICT in Building Design** course at Politecnico di Torino.

## Project Goals

- Building energy performance simulation (EnergyPlus)
- Thermal comfort and energy demand analysis
- Energy signature modelling
- AI-based prediction using LSTM
- Design optimization & Pareto analysis
- Smart building control strategy development
